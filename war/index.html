GitHub Status Bot

GitHub Status Notifications via Google Talk

GitHub Status Bot is a Google Talk Bot that can provide you with GitHub System Status. Optionally, you can also subscribe to get notified via IM in case of any minor or major status issues with GitHub.

Just add githubstatusbot@appspot.com as your Google Talk friend and type help to me.

Powered by GitHub Status API & Google AppEngine
