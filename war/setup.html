GitHub Status Bot

  1. Fire up Google Talk! and add githubstatus@appspot.com as a friend.

  2. Once I am added, understand the language I speak. It is not that complicated and I understand only 7 commands. All you have to do is type the command name and send it to me.

  3. help: Type this anytime to get a list of commands that I understand.
  4. about: In case you want to know who created me. In case you have some create ideas to create other bots, learn how to write one. We bots are very noble. May our tribe increase.

  5. status: Get the current system status (one of good, minor, or major) and timestamp.

  6. last-message: Get the last human communication, status, and timestamp.

  7. messages: Get the most recent human communications with status and timestamp.

  8. subscribe: I can notify you in case there are any major or minor status issues with the GitHub System. I will assume that you are online on your GTalk during that time.

  9. unsubscribe: To remove yourself from any IM notifications by me. This will remove you from getting notified in case there are any minor or major status issues with GitHub Sytstem.
